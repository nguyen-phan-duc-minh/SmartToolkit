# SmartToolkit v1.0.2-beta - Thử nghiệm khép kín

## Thông tin phiên bản
- **Version Name:** 1.0.2-beta
- **Version Code:** 3
- **Build Type:** Internal Testing / Closed Beta
- **AAB Size:** 64.7MB
- **Target SDK:** 35 (Android 15)
- **Min SDK:** 21 (Android 5.0+)

## Mục đích
Phiên bản này được tạo để thử nghiệm khép kín trước khi release chính thức lên Google Play Store.

## Thay đổi từ v1.0.1
- Cập nhật version code từ 2 lên 3
- Đánh dấu là phiên bản beta để phân biệt với release chính thức
- Chuẩn bị cho giai đoạn thử nghiệm khép kín

## Tính năng không đổi
Tất cả 15 công cụ tiện ích vẫn hoạt động như bình thường:
- Calculator (Máy tính)
- Notes (Ghi chú)
- Password Generator (Tạo mật khẩu)
- QR Generator/Scanner
- Tip Calculator (Tính tiền tip)
- Unit Converter (Chuyển đổi đơn vị)
- Age Calculator (Tính tuổi)
- BMI Calculator
- Stopwatch (Đồng hồ bấm giờ)
- Todo List
- Flashlight (Đèn pin)
- Countdown Timer (Đếm ngược)
- Image to Text (OCR)
- Sound Meter (Đo âm thanh)

## Tệp build
- **AAB:** `build/app/outputs/bundle/release/app-release.aab`
- **Debug Symbols:** `build/app/outputs/mapping/release/mapping.txt`
- **Signed:** Có (SmartToolkit Release Key)

## Hướng dẫn thử nghiệm
1. Upload AAB lên Google Play Console
2. Tạo Internal Testing track
3. Thêm tester emails
4. Phân phối cho nhóm thử nghiệm khép kín
5. Thu thập feedback trước khi release chính thức

## Lưu ý
- Đây là phiên bản beta chỉ dành cho thử nghiệm
- Không phân phối công khai
- Report bugs qua internal channels
